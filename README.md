# 🎓 Internship Management System

A full-stack web application to efficiently manage internships, track intern progress, and simplify the admin-intern interaction — built as part of a task assigned by [Internee.pk](https://internee.pk).

## 📌 Features

- ✅ **Role-Based Authentication** (Intern & Admin)
- ✅ **Internship Application Management** (Intern applies, Admin approves)
- ✅ **Admin Dashboard**:
  - Onboard new interns
  - Assign and manage tasks
  - Review and approve internship applications
- ✅ **Intern Dashboard**:
  - View assigned tasks
  - Submit project links/screenshots
  - Track progress in real-time
- ✅ **Task Feedback System** (Admin can approve or give feedback)
- ✅ **MongoDB Integration** for secure data storage
